# Dress Up Game

A simple web-based dress up game where users can drag and drop clothes and accessories onto a character. Built with HTML, CSS, and JavaScript.

## How to Play
- Open `index.html` in your browser.
- Drag clothing and accessory items onto the character to dress them up.
- Mix and match different items for fun combinations!

## Features
- Drag and drop interface
- Multiple clothing and accessory options
- Modern, visually appealing UI
